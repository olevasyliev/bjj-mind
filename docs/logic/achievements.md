# BJJ Mind — Titles & Achievements

---

## Overview

Titles are displayed on the Profile screen as badges.
Earned titles stay forever — they can't be lost.
Each title has: name, icon, unlock condition, rarity.

Rarity: `common` | `rare` | `epic` | `legendary`

## Title Display Rules

- Profile screen shows: all earned titles as badges
- Locked titles shown as greyed out with "???" name (for common/rare) or fully hidden (epic/legendary)
- Most recent 3 earned titles highlighted at top of Profile
- Title earned screen: shown after session summary if new title earned (celebration animation)
